# Frontend criado para a matéria de PW4 (Programação para Web IV) - 02/2023
## Tecnologias utilizadas:
1. React - TS
2. Next.js
3. Axios
4. Styled-components
5. React-toastify
6. Fontawesome

Projeto integrado com a API https://github.com/leticiotte/pw4-api-project, desenvolvida por mim também.
